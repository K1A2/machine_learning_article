# 기사 자동 뷴류 모델 만들기
프로젝트 설명
-----------
기사 전문을 입력하면 기사가 8가지 카테고리중 어디에 속하는지 자동으로 분류해 주는 모델을 직접 학습하여 만들어 보는 프로젝트.
- 프로젝트 기간: 2020년 7월 ~ 2020년 9월

프로젝트 진행 과정
-------
1. 학습에 사용할 기사 크롤링 시작.
    - 2010년 1월 ~ 2020년 5월까지 3일에 한번씩 그 날 작성된 기사를 모두 가져옴.
    - 문화, IT, 경제, 엔터테이먼트, 외국, 정치, 사회, 스포츠 8개의 카테고리에서 각각 기사 수집.
